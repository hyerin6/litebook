# litebook
Social Network Service (Timeline)    

친구와 가족, 같은 관심사를 가진 사람들과 소통할 수 있는     
좋아하는 컨텐츠 업로더를 구독하거나 나의 이야기를 공유할 수 있는 서비스입니다.
